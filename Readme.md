Student Grade Tracker (Java)
📌 Project Overview
This is a simple Java console-based application that helps track student marks.
It allows users to enter marks for multiple students and generates a complete summary report including average, highest, and lowest marks.
🚀 Features
Input marks for multiple students
Store marks using ArrayList
Calculate:
Total marks
Average marks
Highest marks
Lowest marks
Display clean summary report
🛠️ Technologies Used
Java
ArrayList
Scanner (for user input)
